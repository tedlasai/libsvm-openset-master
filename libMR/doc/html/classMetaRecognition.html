Parameters:
scores_to_dropis this object for prediction, if so how many top scores to drop when fitting
fitting_sizetail size for fitting. With small data the defaults are fine.. if you have millions make it larger for better predictions
verboseis the code chatty on errors during fitting, useful for debugging
alphaband for confidence interfals

Use FitLow if your data is such that smaller scores are better.. we'll transform it for you and keep the transform parameters in the class so later calls to W_score or CDF do the right thing. returns 1 for success, <0 for error code
